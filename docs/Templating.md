# Variables in HTML

## User Profile
+ UserName: `{{userName}}`
+ E-Mail: `{{email}}`
+ Date Join: `{{dateJoin}}`
+ Stars Rating: `{{userRating}}`
+ Stars given: `{{starsGiven}}`
+ Stars received: `{{starsRecv}}`
+ Number of Threads: `{{numThreads}}`
+ Number of Replies: `{{numReplies}}`
+ Account Type: `{{accountType}}`

## Thread 
+ Thread Title: `{{title}}`
+ Description: `{{desc}}`
+ Category: `{{category}}`
+ Number of Replies: `{{numReplies}}`
+ Star Rating: `{{threadRating}}`
+ Posted by: `{{user}}`

## Reply
+ Reply ID: `{{replyID}}`
+ ReplyBy (username) : `{{replyBy}}`
+ Text: {{replyText}}
+ Rating:`{{replyRating}}`
